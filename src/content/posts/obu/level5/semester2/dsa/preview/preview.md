## Assessment Information

**Coursework 1 (weighs 20% of the module)**  
Ø Part 1： Implementation of Coursework 1  
Programming and testing implementations on hash tables, linked lists and sorting algorithms, including relevant knowledge.  
Ø Part 2:    A class test of Coursework 1  
Questions related to the implementation of coursework 1.  

**Note:** If there is no corresponding complete source code file submitted on time, the marks may be zero.

Coursework 2 (weighs 30% of the module)  
Ø Part 1： Implementation of Coursework 2  
Programming and testing implementations on binary trees, graphs and path-finding algorithms, including relevant knowledge.  
Ø Part 2:    A class test of Coursework 2  
Questions related to the implementation of coursework 2.  

**Note:** If there is no corresponding complete source code file submitted on time, the marks may be zero.

Final examination(weighs 50% of the module)
- Questions related to the content delivered during the semester, including lectures, practical exercises and coursework.
    - MCQs
    - Short answer questions
    - Programming questions
